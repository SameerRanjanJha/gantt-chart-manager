# Gantt Chart Manager 🗓️

A standalone Python desktop application to create, view, and manage Gantt charts using CSV files.

![image](https://github.com/user-attachments/assets/226053d2-9192-4df7-b67e-3c59d28081d1)


## Features
- Load CSVs with task data (name, start, end, owner)
- Interactive Gantt chart using Matplotlib
- Edit, add, delete tasks with PySimpleGUI
- Drag-adjustable task dates
- Undo/Redo functionality
- Export to CSV

## Installation

```bash
pip install PySimpleGUI matplotlib pandas
